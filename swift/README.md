# ArduinoBluetooth
A small example of data exchange from iOS to Arduino

# Components

* 1 Arduino Uno
* 1 BLE Link XBee
* 1 Arduino XBee Shield
* 3 LEDs (Red, Green, Blue)
* 1 330 ohm Resistor
* 1 Wire
